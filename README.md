### Want to run it locally??
- `git clone https://github.com/A7ryan/golang-static-webserver.git`

- `cd code`

- `go mod tidy`
#### if needed run: `go mod init <your-github-username/repo>`

- `go run main.go`

- Open in Browser: `http://localhost:8080`
